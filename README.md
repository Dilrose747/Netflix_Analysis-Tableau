# Netflix Content Analysis Dashboard | Tableau

## Overview
This project is an interactive Tableau dashboard that analyzes Netflix’s global content library to identify trends in content distribution, ratings, genres, and growth over time. It provides both high-level insights and detailed, title-level exploration through dynamic filtering.

## Features
- **Global Content Map**: Visualizes total movies and TV shows by country  
- **Ratings Distribution**: Shows content split across rating categories  
- **Movies vs TV Shows**: Compares catalog proportions using bubble visualization  
- **Top Genres**: Highlights the most frequent genre combinations  
- **Year-wise Growth**: Displays content trends over time for movies and TV shows  
- **Interactive Metadata Panel**: Updates title details such as rating, duration, release year, and genre on selection

## Tools & Technologies
- **Tableau** – Dashboard design and interactive visualizations  
- **Dataset** – Netflix Movies and TV Shows (public dataset)


## How to Use
1. Download or clone this repository  
2. Open the `.twbx` file in Tableau Desktop or Tableau Public  
3. Use filters or click on any title to explore detailed content metadata

## Key Insights
- Movies dominate Netflix’s catalog compared to TV shows  
- Content growth accelerates significantly after 2015  
- Documentaries, international dramas, and stand-up comedy are among the most common genres  
- The U.S. and India contribute the highest number of titles
